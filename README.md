# Request Management Frontend

This is a test application with Vue.js as Frontend for simple Contacts list

## 1. Fronted App

* Javascript
* Vue.js 2.6.10
* Vuex 3.1.1
* Axios 0.19.0
* Vuetify 2.0.5
* Webpack 3.7.1
* Node.js 8.11.3
* NPM 5.6

I strongly recommend to use [Node Version Manager](https://github.com/creationix/nvm) for building the application

## 2. Build Setup

## Project setup
```
# install dependencies
npm install

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Run your tests
npm run test

# Lints and fixes files
npm run lint
```
